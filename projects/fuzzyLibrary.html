Fuzzy-Library: librería Python para modelado y evaluación de sistemas de lógica difusa
Reto
Definir, validar y evaluar sistemas de lógica difusa suele implicar repartir la lógica entre clases ad hoc, fórmulas aisladas y reglas difíciles de mantener. El objetivo del proyecto es concentrar ese trabajo en una librería reutilizable que permita modelar variables lingüísticas, conjuntos difusos y reglas con una estructura consistente.
Solución
Librería modular en Python con soporte para conjuntos difusos trapezoidales, variables lingüísticas, proposiciones atómicas y compuestas, parser de reglas IF ... THEN ... y evaluación de reglas para modelos Mamdani, TSK y Cumulative TSK. El proyecto incluye además un evaluador de fórmulas para consecuentes TSK y una batería de tests unitarios sobre parser, tipos y componentes base.
Estado actual
La base del núcleo, el parser y la evaluación individual de reglas ya están implementadas. En el motor de inferencia, la agregación Mamdani está desarrollada, mientras que la agregación para TSK, Cumulative TSK y los algoritmos de mezcla entre modelos siguen pendientes.
