Chatbase Integrado
Chatbot del portfolio
Asistente contextual integrado en la web con Chatbase
Reto
Añadir una vía de ayuda rápida dentro del portfolio sin construir todavía un backend propio, pero manteniendo una integración limpia y fácil de activar o desactivar.
Solución
Se integró el widget de Chatbase mediante su snippet de JavaScript, aislándolo en un archivo propio y cargándolo solo en las páginas necesarias para no mezclarlo con la lógica principal del portfolio.
Estado actual
Proyecto integrado y funcionando en la web. Sirve como demostración de integración de herramientas de terceros y como base para una futura versión propia con backend y modelo local o remoto.
