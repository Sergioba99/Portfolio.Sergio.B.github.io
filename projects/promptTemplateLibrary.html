Prompt Template Library
Gestor de plantillas YAML para LLMs con SQLite y sincronización de metadatos
Reto
Centralizar la creación de prompts y plantillas reutilizables para modelos de lenguaje sin perder orden, trazabilidad ni consistencia entre archivos, metadatos y estructura interna.
Solución
Se plantea una aplicación de escritorio con PySide6 para editar, generar y previsualizar prompts a partir de plantillas YAML, apoyada por SQLite para indexar metadatos y por sincronización basada en contenido para mantener alineados archivos y base de datos.
Estado actual
Proyecto en desarrollo. La base de documentación ya está definida y se está avanzando en la primera implementación de la herramienta de escritorio.
